# July 18 performance campaign

This directory is the public, scrubbed package for the July 18 GB10 performance and quality work. Start with [`DAY.md`](DAY.md), then follow each topic's README.

The package intentionally separates:

- kernel microbenchmarks from served throughput;
- same-prompt comparisons from prompt changes;
- completed measurements from staged-but-not-yet-adopted optimizations;
- quality scores from model footprint and training-progress receipts.

All paths are relative or use `$HOME`; worker names, private addresses, internal task IDs, credentials, and tokens have been removed. Public attribution: **@banana_bae**.

Per-topic `MANIFEST.sha256` files are the publication integrity authority. Where an original campaign MD5 exists it is retained in the topic receipt; SHA-256 is used for all repository payloads and manifests.
